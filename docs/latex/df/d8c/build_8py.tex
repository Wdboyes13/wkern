\doxysubsection{Detailed Description}
Entry point for the OS kernel build system. 

This script coordinates the entire build process, as well as cleanup, running, and testing workflows.\hypertarget{build_8py_autotoc_md18}{}\doxysubsubsection{\texorpdfstring{Usage\+:}{Usage\+:}}\label{build_8py_autotoc_md18}

\begin{DoxyItemize}
\item {\ttfamily ./build.py} \+: Configure (if needed) and build the project
\item {\ttfamily ./build.py -\/clean} \+: Clean build artifacts
\item {\ttfamily ./build.py -\/git} \+: Clean and perform Git-\/specific cleanup
\item {\ttfamily ./build.py -\/run} \+: Run the built image using QEMU
\item {\ttfamily ./build.py -\/test} \+: Re-\/run \doxylink{configure_8py}{configure.\+py} to verify system setup
\item {\ttfamily ./build.py -\/loc} \+: Count lines of code via {\ttfamily loc.\+sh}
\item {\ttfamily ./build.py -\/mkd} \+: Create and format a blank FAT16 disk image 
\end{DoxyItemize}
