build.\+main (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



CLI entrypoint for \doxylink{build_8py}{build.\+py}. 

Parses arguments and dispatches to the correct build system action\+:
\begin{DoxyItemize}
\item No args\+: Build after configuring if necessary
\item {\ttfamily -\/clean}\+: Clean build output
\item {\ttfamily -\/git}\+: Clean output and call {\ttfamily git()} (likely to prepare for commit)
\item {\ttfamily -\/run}\+: Run the kernel via QEMU
\item {\ttfamily -\/test}\+: Validate environment with \doxylink{configure_8py}{configure.\+py}
\item {\ttfamily -\/loc}\+: Run {\ttfamily loc.\+sh} to count lines of code
\item {\ttfamily -\/mkd}\+: Create and format a blank FAT16 image using mkfs.\+fat 
\end{DoxyItemize}
