\doxysubsection{Detailed Description}
Defines file organization and module structure for the build system. 

This file contains lists of source files (C and ASM), grouped into modules. It also defines output targets, build metadata, and clean targets used by other parts of the build system.

The build system uses this file to\+:
\begin{DoxyItemize}
\item Track all C source files ({\ttfamily SRCS})
\item Track all NASM source files ({\ttfamily NASMSRCS})
\item Define how C source files are grouped into logical modules ({\ttfamily MODULES})
\item Generate output ELF files and ISO builds
\item Determine which object files need to be cleaned ({\ttfamily CLEANTARGS})
\end{DoxyItemize}

This file is imported by other scripts such as {\ttfamily \doxylink{build_8py}{build.\+py}}, {\ttfamily \doxylink{brules_8py}{brules.\+py}}, and {\ttfamily \doxylink{frules_8py}{frules.\+py}} to control compilation, linking, and packaging.
