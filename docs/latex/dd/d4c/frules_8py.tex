\doxysubsection{Detailed Description}
Build system utility rules for formatting, Git, run, and header validation. 

This file defines helper routines used during the build pipeline\+:
\begin{DoxyItemize}
\item Code formatting with {\ttfamily clang-\/format}
\item Auto-\/committing via Git
\item Running the kernel in QEMU
\item Validating the ELF binary with {\ttfamily grub-\/file} for multiboot support
\end{DoxyItemize}
